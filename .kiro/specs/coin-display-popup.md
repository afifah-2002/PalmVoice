# Coin Display Popup Specification

## Trigger
- Tap treasure chest icon on Home screen
- Tap treasure chest icon on Pet screen

## Popup Content

### Layout
- Centered modal on screen
- Background: Semi-transparent dark (matches theme)
- Border: 2px solid white
- Padding: 20px

### Content (Top to Bottom)
- Treasure chest icon (large, ~60px)
- Coin count: "X COINS" (pixelated font, large)
- Button: "PLAY TO WIN MORE!" (below coin count)
- X button: Top right to close

### Button Action
- Tap "PLAY TO WIN MORE!": Opens fruit catching game
- Tap X or outside popup: Closes popup

## Style
- Pixelated animated font
- Theme colors
- Simple fade-in animation on open
