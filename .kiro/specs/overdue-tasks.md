# Overdue Tasks

## Detection
- Task due date < current date/time
- Check on app open and every minute

## Display
- Red "!" icon next to due date
- Date text turns red

## Interaction
- Tap "!" or overdue task
- Show popup: "OVERDUE"
- Options:
  - "Mark as Completed"
  - "Extend Due Date"
- Style: Pixelated, theme colors, animated

## Extend Due Date
- Opens calendar picker
- User selects new date
- "!" removed after extension
