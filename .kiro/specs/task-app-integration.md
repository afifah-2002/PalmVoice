# Task App Integration

## Add To Feature
- Each task shows "+" button on right
- Tap "+": Opens scrollable modal

## Modal Content
- Title: "ADD TO APP"
- Grid: 4 columns, scrollable
- Icons from: assets/icons/
- Apps: reminders, alarm, gmail, canvas, outlook, fitness, calendar, whatsapp, phone, zoom, teams, gmeet, paypal, amazon, uber

## Selection
- Tap icon: Toggles selection (highlight/checkmark)
- Multiple selection allowed
- "DONE" button saves and closes

## Display
- Selected icons appear next to task title
- Small icons in a row

## Action
- Tap icon on task: Opens that app via URL scheme
