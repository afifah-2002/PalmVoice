# PalmVoice - Demo Checklist

## Pre-Demo Setup
- [ ] App running on device/simulator
- [ ] Volume turned up for sound effects
- [ ] Calendar permissions granted (for sync demo)
- [ ] Clear any test tasks for clean demo

## Demo Script (5 minutes)

### 1. Introduction (30 seconds)
**Say**: "PalmVoice is a voice-to-task app with a nostalgic Palm OS aesthetic"

**Show**:
- Main task list screen
- Point out monochrome design
- Courier font
- Black borders

### 2. Core Feature: Voice Recording (1 minute)
**Say**: "Let me show you how easy it is to add tasks"

**Do**:
- [ ] Tap "NEW TASK" button
- [ ] Show recording screen
- [ ] Tap gray circle to start recording
- [ ] Point out red pulsing animation
- [ ] Show timer counting up
- [ ] Tap to stop or press "DONE"
- [ ] Show "PROCESSING..." state
- [ ] Return to task list with new tasks

**Highlight**:
- Simple one-tap recording
- Visual feedback (pulsing, timer)
- Automatic task extraction

### 3. Smart Features (1 minute)
**Say**: "The app intelligently parses tasks and dates"

**Show**:
- [ ] Point out tasks with different formats
- [ ] Show due dates (TODAY, TOMORROW, MM/DD)
- [ ] Explain date detection ("tomorrow at 3pm", "friday", etc.)

**Highlight**:
- Natural language date parsing
- Variety of task types (EMAIL, CALL, BUY, MEETING)
- Smart categorization

### 4. Task Management (1 minute)
**Say**: "Managing tasks is intuitive"

**Do**:
- [ ] Tap a task to toggle completion
- [ ] Point out sound effect
- [ ] Show strikethrough and gray color
- [ ] Long press a task
- [ ] Edit the text in the dialog
- [ ] Save changes

**Highlight**:
- One-tap completion
- Sound feedback
- Easy editing with long press
- Palm OS styled dialog

### 5. Advanced Features (1 minute)
**Say**: "PalmVoice integrates with your device"

**Do**:
- [ ] Tap "SYNC" button
- [ ] Show calendar permission (if needed)
- [ ] Show success message
- [ ] (Optional) Open calendar app to show exported tasks

**Highlight**:
- Calendar integration
- Persistent storage
- Cross-app functionality

### 6. About & Credits (30 seconds)
**Say**: "This entire app was built with Kiro AI"

**Do**:
- [ ] Tap "?" button
- [ ] Scroll through About screen
- [ ] Point out Kiro credits
- [ ] Show feature list

**Highlight**:
- Built with AI assistance
- Complete feature set
- Professional polish

## Common Questions & Answers

**Q: Does it really use AI?**
A: Currently uses smart mock data. Real OpenAI integration is implemented and ready to enable with an API key.

**Q: How long did this take to build?**

**Q: Can I use this for real?**
A: Yes! Enable real AI, add your API key, and it's production-ready.

**Q: What makes it "Palm OS"?**
A: Monochrome colors, Courier font, 2px borders, no shadows/gradients, button inversion - faithful to Palm Pilot UI.

A: React Native, Expo, TypeScript, AsyncStorage, Expo AV, Expo Calendar.
